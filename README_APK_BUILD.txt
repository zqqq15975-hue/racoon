- 4개 항목 카드에 “분석완료 100%” 표시
- 회차별 판정 알림 팝업 1회만 표시
- Analyzer 레벨 + UI 레벨에서 중복 팝업 이중 차단
- Selenium/Tkinter/XPath 클릭 제거 유지
- requests로 HTML 결과표 직접 파싱 + API 보조분석
